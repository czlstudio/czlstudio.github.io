6.事件传参与多事件处理函数

  • 小结

    1.@click=”handleClick(msg)”

    2.@click=”handleClick(mag, $event)”

    3.@click=”handleClick1(), handleClick2()”

7.事件相关的修饰符

  • 事件修饰符 可以组合使用 ( @click.once.prevent )
    1. .stop // event.stopPropagation() 停止冒泡。
    2. .capture // 捕获事件,现在该元素中处理,之后交给子元素处理,与事件冒泡相反。
    3. .self // 只处理发生在本元素上的事件。
    4. .once // 事件只处理一次

8.处理键盘、鼠标事件的修饰符

  • 按键修饰符 ( @keyup.page-down )

    1. .enter // 回车键

    2. .tab // tab 制表符

    3. .delete // 退格键

    4. .esc // ESC 键

    5. .up // 上键

    6. down // 下键

    7. .left // 左键

    8. .right // 右键

      • 更多键盘修饰符

  • 常见组合键修饰符

    1. .ctrl // ctrl 键
    2. .alt // alt 键
    3. .shift // shift 键
    4. .meta // Windows: win 键

  • 精确组合键

  • 鼠标修饰符 ( @mousedown.left=”handleMouseClick” )
    1. .left // 鼠标左键
    2. .middle // 鼠标中键, 或鼠标滚轮按下时
    3. .right // 鼠标右键

9.表单输入相关的修饰符 ( v-model )

​ 1. .lazy // 失去焦点才获取数据 -> @change

​ 2. .number // 获取数字数据

​ 3. .trim // 去除头 尾空格
