  • 总结

    1. 初始化 Vue项目,本地引入/CDN。
    2. 调用 Vue.createApp() 创建应用,在调用 mount() 挂载到 HTML 元素上。
    3. Vue 配置对象可定义 data, 并使用 v-bind 或 “:” 绑定 数据到模板。
    4. data 数据是响应式的,模板会根据 data 数据的变化而变化。
    5. 展示列表数据. 根据条件展示数据。
    6. Vue 事件处理,使用 v-on 或 @ 绑定事件监听。
    7. 使用 methods 定义函数。
    8. 定义计算属性. 监听器。
    9. 计算数属性. 监听器. methods 的区别。
    10. 处理表单及 v-model 双向绑定。
    11. 绑定数据到 style 和 class 中。
